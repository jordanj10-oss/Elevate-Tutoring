# Elev8 Tutoring — iOS App (Capacitor)

This directory contains the native iOS Xcode project generated by Capacitor for the Elev8 Tutoring app.

## Prerequisites

- macOS with Xcode 15+ installed
- Node.js 18+
- A valid Apple Developer account ($99/year) for App Store distribution

## Setup

```bash
# 1. Install dependencies (already done)
cd /home/team/shared/tutoring-platform
npm install @capacitor/core @capacitor/cli @capacitor/ios

# 2. Initialize Capacitor (already done)
npx cap init "Elev8 Tutoring" "com.elev8tutoring.app"

# 3. Add iOS platform (already done)
npx cap add ios

# 4. Copy web assets (already done)
npx cap copy
```

## Build for App Store

```bash
# 1. Build the web app
npm run build

# 2. Sync web code to native project
npx cap copy

# 3. Open in Xcode
npx cap open ios

# 4. In Xcode:
#    - Select a development team in Signing & Capabilities
#    - Set bundle identifier: com.elev8tutoring.app
#    - Build with Product > Archive
#    - Upload to App Store Connect via Distributor > App Store Connect
```

## Project Structure

```

├── App/
│   ├── App/              # Native Swift code

│   │   └── ViewController.swift
│   ├── App.xcodeproj/    # Xcode project file
│   ├── Podfile           # CocoaPods (if needed)
│   └── public/           # Copied web assets (auto-generated by cap copy)
├── capacitor.config.json # Capacitor configuration

```

## Notes

- The `ios/` directory is generated and should be committed to the repo
- After updating the web app, always run `npx cap copy` to sync changes
- For live reload during development, use `npx cap run ios -l --external`
- Splash screen and icon customization can be done via Capacitor plugins
